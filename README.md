# Mini Library System (C++ OOP)

## Описание
Това е проста библиотечна система, реализирана с добри ООП практики в C++.

Проектът демонстрира:
- капсулация
- валидация на данни
- Rule of 5 (копиране/преместване)
- статични членове
- конструктори и деструктори
- const-коректност

Всички класове са в отделни `.h` файлове:
- `Author.h`
- `Book.h`
- `Member.h`
- `Loan.h`
- `Library.h`
- `main.cpp`

## Компилация
```bash
g++ -std=gnu++17 -Wall -Wextra -O2 -o library main.cpp
